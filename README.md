# Eagle Bank API

A REST API for a fictional bank built with Spring Boot 4, Java 25, and JWT authentication.

## Tech stack

- Java 25, Spring Boot 4.0.5, Maven
- Spring Security
- Spring Data JPA, H2 in-memory database

## How to run

```bash
./mvnw spring-boot:run
```

The API starts on `http://localhost:8080`.

## How to test

```bash
./mvnw test
```
